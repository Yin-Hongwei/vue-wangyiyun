## Function

音乐来源是调用别人写好的API，前端用vue来实现，具体有页面的展示，音乐搜索，音乐播放及相关控制，歌词显示。



## Technologies

vue + vue-router + vuex + axios + mint-ui



## Install

下载本项目到本地

```
git clone https://github.com/Yin-Hongwei/vue-wangyiyun.git
```

安装依赖

```
npm install
```

启动项目

```
npm run dev
```
