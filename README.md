## 功能

- 页面的展示，
- 音乐搜索，
- 音乐播放
- 歌词显示



## 技术栈

vue + vue-router + vuex + axios + mint-ui



## 安装

下载本项目到本地

```
git clone https://github.com/Yin-Hongwei/vue-wangyiyun.git
```

安装依赖

```
npm install
```

启动项目

```
npm run dev
```
