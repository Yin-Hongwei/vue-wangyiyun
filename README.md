# 🎵🎵🎵基于 vue 仿写网易云音乐webapp



## 1功能概述

音乐来源是调用别人写好的API，前端用vue来实现，具体有页面的展示，音乐搜索，音乐播放及相关控制，歌词显示。



## 技术栈：

vue + vue-router + vuex + axios + mint-ui
